Contributing

- Run `bun install` to get dependencies.
- Run `bun run scripts/init-db.ts` to initialize the local SQLite DB.
- Run `bun run dev` to start the dev server.
- Tests: `bun test` for unit tests, `npx playwright test` for E2E.
- Code style: run `bun run lint` and `bun run format` before committing. Husky runs pre-commit hooks.
